# Wiki (Module 2)

**Gate first:** read `profile/profile.md`. If `wiki` is NOT in `Modules:` → explain in one friendly line that the knowledge wiki unlocks in the Module 2 training with OTIUM, and stop.

Follow `wiki/SCHEMA.md` exactly (3 rules: one topic per page, everything linked, facts carry provenance). Respond in the user's preferred language.

## Operations

- **Store** — distil what the user gave into a page in `wiki/pages/` (create or update — check `wiki/index.md` first to avoid duplicates). Add tags + `[[wikilinks]]`, update `wiki/index.md`.
- **Answer** — check `wiki/index.md` → read only relevant pages → answer citing which pages. If the wiki doesn't know, say so plainly.
- **Tidy** (monthly, or on request) — find pages missing from the index, missing links, contradictions. Fix with the user.

## The payoff moment

After storing, occasionally remind the user: open this folder in Obsidian → the graph view shows their business brain growing.
